# References

This directory is used only for references that are directly used in project notes, reports, code, or blog posts.

The full reading database is maintained separately in a private Google Drive spreadsheet.

## Policy

- Do not upload copyrighted PDFs.
- Do not duplicate the full private reading spreadsheet here.
- Record only sources that are directly connected to project outputs.
- Weekly notes and milestone reports should include their own reference sections.

## Examples of sources to record here

- EEG/BCI papers directly used in analysis notes
- MNE or OpenBCI documentation used in code
- signal-processing references used for filtering or PSD analysis
- books or papers cited in milestone reports
