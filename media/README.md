# Media

This directory stores media files related to the project.

## Examples

- demo videos
- screen recordings
- OpenBCI GUI screenshots
- robot-control demonstration clips
- presentation media
- supplementary visual materials

## Directory Structure

- `demos/`: Demonstration videos
- `screenshots/`: Screenshots from tools, interfaces, or experiments

## Policy

- Do not store raw EEG data here.
- Do not store large media files unless necessary.
- For large files, use external storage and document the link or file location.
- Avoid uploading identifiable participant videos or images without explicit consent.

## Naming Convention

Use descriptive filenames with session numbers when possible.

Examples:

- `session-10-openbci-gui-connection-test.mp4`
- `session-22-eeg-to-arduino-demo.mp4`
- `session-30-final-demo-video.mp4`
