# bci-eeg-robot-control
# BCI-based EEG SIgnal Analysis and Robot Conrol

This repository documents my one-year research preparation project on EEG signal analysis, BCI, and simple robotic control.	
	
## Goals	
- Learn EEG/BCI fundamentals	
- Build a Python-based EEG analysis pipeline	
- Implement a simple EEG-to-Arduino control loop	
- Record progress in a research-oriented format	
	
## Structure	
- docs/: project-overview, timeline, milestones
- weekly-notes/: weekly session notes
- src/: code	
- results/: figures and outputs	
- reports/: summaries and reports
- media/: screenshots
