# Reports

This directory contains synthesized reports produced during the EEG-BCI robot control project.

Unlike weekly notes, which document each session chronologically, reports summarize progress across multiple sessions or experimental phases.

## Report Types

- Milestone reports
- Experiment reports
- Analysis reports
- Final project report

## Current Reports

| Report | Description |
|---|---|
| `milestone-01-session-01-to-03.md` | Summary of Sessions 01–03, covering conceptual foundation, EEG interpretation, feature candidates, and next actions |

## Directory Role

- `weekly-notes/`: chronological session logs
- `docs/`: concept notes and blog archives
- `reports/`: synthesized project reports
- `results/`: analysis outputs, figures, and quantitative results
- `src/`: code
