# Weekly Notes

This directory contains session-by-session study and research notes.

Each note records what was studied, what was understood, what remains unclear, and what should be done next.

## Purpose

Weekly notes are not diaries.

They are research records that document:

- objectives
- study contents
- key takeaways
- personal understanding
- unclear points
- next actions
- outputs
- references

## Naming Convention

Use the session number and date.

Examples:

- `session-01-260328.md`
- `session-02-260418.md`
- `session-03-260425.md`
- `session-04-260502.md`
- `session-05-260509.md`

## Suggested Format

# Session XX — Title

## 1. Objectives

## 2. What I Studied

## 3. Key Takeaways

## 4. My Understanding

## 5. Questions / Unclear Points

## 6. Next Actions

## 7. Outputs

## 8. References

## Principle

Each weekly note should help reconstruct the research process later.

A good weekly note should answer:

1. What was the goal of this session?
2. What was actually done?
3. What changed in my understanding?
4. What concrete output was produced?
5. What should happen next?
