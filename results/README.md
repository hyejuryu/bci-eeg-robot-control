# Results

This directory stores analysis outputs and result summaries.

## Examples

- CSV summary files
- calculated band power values
- feature comparison tables
- model or threshold evaluation results
- log files
- analysis summaries

## Difference from Figures

- `figures/`: visual outputs such as plots, diagrams, and images
- `results/`: numerical or structured outputs such as CSV, TXT, JSON, or summary tables

## Directory Structure

Use session-based folders when possible.

Examples:

- `session-05/`
- `session-06/`
- `session-07/`
- `session-08/`

## Naming Convention

Use descriptive filenames.

Examples:

- `session-07-alpha-beta-power-summary.csv`
- `session-08-eyes-open-closed-comparison.csv`
- `session-22-threshold-trigger-log.csv`
- `session-29-control-accuracy-summary.csv`

## Policy

Results should be reproducible from scripts or notebooks whenever possible.

Each result file should be traceable to:

- input data
- analysis script or notebook
- session number
- date
- analysis parameters
