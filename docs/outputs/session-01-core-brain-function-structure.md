# Core Brain Function Structure
Related study note: [Session 01](../../weekly-notes/session-01-260328.md)

## session-01-core-brain-function-structure

| Domain | Main related region(s) | Why it matters here |
|---|---|---|
| Voluntary movement | Frontal motor-related regions, PMA, M1 | Candidate basis for intention-related control |
| Somatosensory processing | Parietal sensory cortex and association areas | Important for understanding sensorimotor hierarchy |
| Sensorimotor integration | PPC, premotor regions, M1 | Helps frame movement as sensing–integration–planning–execution |
| Attention | Frontal-striatal and prefrontal-related networks | Strong candidate domain for threshold-based EEG control |
| Neural signaling | Neurons, synapses, neurotransmitters | Fundamental basis for interpreting EEG signals |
